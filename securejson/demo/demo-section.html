🧪 Try SecureJSON Live

Step 1: Generate Key Pair

Terminal
$ securejson generate-key --algo es256 --prefix demo
🔑 Generating ES256 key pair...
✅ Private key saved to: demo-private.pem
✅ Public key saved to: demo-public.pem
🔐 Algorithm: ES256

Step 2: Sign Your Data

Terminal
$ securejson sign --key demo-private.pem --file nft-metadata.json --out signed.json --algo es256
📁 Signing file...
🔐 Using algorithm: ES256
✅ Signing complete!
📁 File: signed.json

Step 3: Verify Signature

Terminal
$ securejson verify --key demo-public.pem --file signed.json
📁 Verifying file...
🔐 Using algorithm: ES256
✅ Signature verified!
📦 Payload:

}
